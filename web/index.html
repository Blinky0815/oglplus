OGLplus: a C++ wrapper for OpenGL©

(version 3 and higher)

Introduction

OGLplus is a open-source header-only library which implements a thin object-oriented facade over the OpenGL® (version 3 and higher) C-language API. It provides wrappers which automate resource and object management and make the use of OpenGL in C++ safer and easier. Read more...

Features

  • Automatic resource management
  • Type safety
  • Error handling and diagnostic
  • Interoperability with OpenGL C API
  • Additional CG-related classes
  • Optional description of objects and resources
